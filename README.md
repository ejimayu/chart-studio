# Chart Studio

Chart Studio is a quiet chart-making tool for Japanese presentation assets.

It runs as a static GitHub Pages site with no build step.

## Features

- Paste CSV data
- Switch between bar and line charts
- Customize title, subtitle, source, colors, typography, alignment, and spacing
- Export PNG, SVG, and reusable standalone HTML
- Uses Gen Interface JP for Japanese UI and exported chart typography
