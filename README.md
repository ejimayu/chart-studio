# Chart Studio

Chart Studio turns pasted data into presentation-ready chart assets.

It runs as a static GitHub Pages site with no build step.

## Features

- Paste CSV data
- Switch between bar and line charts
- Customize title, subtitle, source, accent color, and background color
- Export a 1920:1080, 1200:630, or 1080:1080 PNG
- Export reusable standalone HTML

## GitHub Pages

Publish from the `main` branch root:

1. Open repository settings.
2. Go to Pages.
3. Set source to `Deploy from a branch`.
4. Select `main` and `/root`.

The app will be available at:

```text
https://<username>.github.io/chart-studio/
```
